# PlayFree Games - Multi-language HTML5 Game Hub

A multi-language HTML5 game aggregation site targeting Spanish and Portuguese markets.

## Features
- 3 languages: English, Spanish, Portuguese
- Category-based game browsing
- Search functionality
- Ad placeholder slots (ready for Google AdSense)
- SEO optimized (sitemap, robots.txt, Open Graph)
- Responsive design
- Auto language detection from browser

## Deploy to Vercel
1. Push to GitHub
2. Import repo in Vercel
3. Deploy automatically

## Tech Stack
- Next.js 15
- Tailwind CSS 4
- TypeScript
